# Chat Feature

Self-contained chat functionality.

## Structure

- **screens/** - Chat UI screens (ChatScreen, etc.)
- **components/** - Reusable chat components (ChatMessage, ChatInput, etc.)
- **hooks/** - Chat-specific hooks (useChat, useChatHistory, etc.)
- **services/** - Business logic (message formatting, history management)
- **types/** - TypeScript types (Message, Chat, etc.)
